# Security

Please do not open public issues containing secrets, private vault content, or sensitive transcripts.

## Reporting

For security or privacy concerns, open a private GitHub security advisory if available for this repository, or contact the maintainer through their GitHub profile.

## Supported Scope

This repository provides:

- Skill instructions.
- Local validation and packaging scripts.
- A lightweight secret scanner for common accidental leaks.

The scanner is a safety net, not a full security audit. Review changes manually before publishing.

## Responsible Use

When using the skill:

- Keep raw transcripts out of long-term memory unless there is explicit consent and a clear need.
- Remove secrets before saving notes.
- Treat Graphify output as derived data and review what gets indexed.
